# HP54602B remote osciloscope controller
This repo contains all developments of my custom scope controll softwar, later to be integrated inside scomesh

## Modularity
The whole software package is thought of to be as modular as possible, the main reason behid that is it beeing a planed module for scomesh(a node based realtime data processing pipeline), inside of which the nodes will need to have access to every single detail inside the controller and even the UI needs to be fully modular to allow the construction of custom control panels(another crucial feature of scomesh)
